# Установщик RAP ATLAS

Здесь лежат открытые исходники Windows-приложения, установщика и деинсталлятора. Пользователям нужен только готовый файл `release\УСТАНОВИТЬ RAP ATLAS.exe`.

## Что получает пользователь

- один установщик без Node.js, pnpm и исходников;
- понятное окно с полным журналом установки;
- включённые по умолчанию пункты «Создать ярлык на рабочем столе» и «Запустить после установки»;
- установку без прав администратора в `%LOCALAPPDATA%\Programs\RAP ATLAS`;
- ярлыки приложения и удаления в меню «Пуск»;
- обычное удаление через «Установленные приложения» Windows;
- отдельный деинсталлятор с полным журналом удаления.

Приложение поднимает сервер только на `127.0.0.1`. После закрытия последней вкладки RAP ATLAS сервер завершает работу. Повторный запуск с ярлыка снова запускает сервер и открывает сайт.

Каждый запуск получает случайный одноразовый токен. Служебные команды сервера принимаются только с этим токеном и только с локальной страницы RAP ATLAS. Одновременно работает одна копия приложения; повторный запуск просто открывает уже запущенную.

Избранное, недавнее и тема лежат в папке установленного приложения. Деинсталлятор удаляет их вместе с приложением, ярлыками и регистрационной записью.

## Сборка

В корне проекта должны быть установлены зависимости (`pnpm install`). Затем запусти:

```powershell
.\installer\build-release.ps1
```

Для сборки нужны Node.js, Python с Pillow и стандартный компилятор .NET Framework из Windows. Иконка берётся из `installer\assets\RAP ATLAS.png`.

Единый номер версии читается из корневого `package.json`. После сборки рядом с установщиком создаётся `release-manifest.json` с версией и SHA-256. Собранные файлы не содержат путей профиля разработчика.

Проверка чистой установки и удаления описана в `COMPATIBILITY-CHECKLIST.md`. Телеметрии, фонового обновления и автозапуска вместе с Windows в приложении нет.
