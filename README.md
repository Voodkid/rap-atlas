## Что это

RAP ATLAS помогает найти и разобрать направление даже без знания жанровых названий. Можно искать по артисту, продюсеру, басу, драмке или характеру мелодии, а затем посмотреть, как термин связан с соседними сценами и стилями.

Текущая версия — **0.3.0**:

- 253 карточки в 12 больших ветках;
- 250 подробно проверенных карточек;
- 1061 источник;
- 1246 треков и других примеров для проверки на слух;
- отдельные тип термина, распространённость и степень проверки;
- связи «выросло из», «пересекается», «часто путают» и «альтернативное название».

Три оставшиеся карточки сохранены в базе, но прямо отмечены как непроверенные.

## Что умеет

- искать жанр, артиста, продюсера или слышимый признак;
- подбирать звук по басу, драмке, мелодии и общей подаче;
- показывать короткий разбор, продюсерские детали и историю термина;
- сравнивать до трёх карточек;
- сохранять избранное и недавние просмотры локально;
- строить короткие маршруты по большим веткам;
- объяснять, почему результат попал в подбор;
- отделять жанры от сцен, коллективов, продюсерских школ и type-beat-тегов.

`Редактор карточки` пока является черновиком интерфейса для будущей версии сайта на хостинге. Он не изменяет общую базу и ничего не публикует.

## Установка на Windows

1. Откройте страницу [последнего релиза](https://github.com/Voodkid/rap-atlas/releases/latest).
2. Скачайте `RAP-ATLAS-Setup.exe`.
3. Запустите установщик и выберите, нужен ли ярлык и автоматический запуск.

Программа работает локально, не требует аккаунта, не отправляет телеметрию и не остаётся висеть в фоне после закрытия последней вкладки. Удаление доступно через список установленных приложений Windows и ярлык в меню «Пуск».

## Тестовая версия для macOS

1. Скачайте `RAP-ATLAS-macOS-Beta.dmg` из [тестового выпуска](https://github.com/Voodkid/rap-atlas/releases/tag/v0.3.0-macos-beta.1).
2. Откройте образ и перетащите `RAP ATLAS` в `Applications`.
3. При первом запуске нажмите на приложение правой кнопкой и выберите «Открыть».

Сборка универсальная: она рассчитана на Apple Silicon и Intel. Версия пока не подписана сертификатом Apple, поэтому macOS показывает стандартное предупреждение. Инструкция и удаление с полным списком файлов находятся внутри образа.

## Запуск из исходников

Понадобятся Node.js 22.13 или новее и pnpm 11.

```bash
pnpm install
pnpm dev
```

Для Windows также сохранён однокнопочный запуск через `ОТКРЫТЬ RAP ATLAS.cmd`.

Проверка проекта:

```bash
pnpm lint
pnpm test
```

## Как устроена база

- `app/atlas-data.ts` — исходное дерево из 253 карточек и функции карты;
- `app/research-data.ts` — модель исследований и 50 карточек, подготовленных вручную;
- `app/research-generated.ts` — ещё 200 карточек, собранных из исследовательских отчётов;
- `Research/` — отчёты, аудит и повторяемая сборка данных;
- `app/AtlasApp.tsx` — интерфейс и пользовательские функции;
- `installer/` — исходники установщика и локального сервера;
- `tests/` — проверки базы и серверного рендера.

## Сборка установщика

На Windows:

```powershell
./installer/build-release.ps1
```

Готовый файл и `release-manifest.json` появятся в `installer/release/`. Папка не хранится в Git: установщик публикуется отдельным файлом в GitHub Releases.
