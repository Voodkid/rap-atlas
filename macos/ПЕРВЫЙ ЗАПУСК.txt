RAP ATLAS для macOS — тестовая версия
=====================================

Установка:
1. Перетащи RAP ATLAS в папку Applications.
2. Открой Applications.
3. Нажми на RAP ATLAS правой кнопкой мыши и выбери «Открыть».
4. В появившемся окне ещё раз нажми «Открыть».

Обычный двойной клик может быть заблокирован, потому что тестовая версия пока
не подписана платным сертификатом Apple. Исходный код и сборка открыты на GitHub.

Если пункта «Открыть» нет:
Системные настройки → Конфиденциальность и безопасность → Открыть всё равно.

Удаление:
Запусти файл «Удалить RAP ATLAS.command» из этого образа. Он заранее покажет
полный список файлов и удалит только приложение и его локальные настройки.

Ошибки: https://t.me/RAPATLAS/33
Исходный код: https://github.com/Voodkid/rap-atlas
